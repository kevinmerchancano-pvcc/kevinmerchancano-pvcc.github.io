All About Github

Questions Answers Terms Definitions
What is version control system? Is the practice of tracking and managing changes to software code Repository It's a place where you can store your code, your files, and each file's revision history
What is Git? Is a version control system used for tracking changes in computer files Commit Is a snapshot of the hierarchy (Git tree) and the contents of the files (Git blob) in a Git repository
What is Github? Is an increasingly popular programming resource used for code sharing Fork Is a new repository that shares code and visibility settings with the original “upstream” repository
What is the difference between Git and Github? Git is a version control system that allows developers to track changes in their code. GitHub is a web-based hosting service for git repositories Push Is how you transfer commits from your local repository to a remote repo
Who started Github and how was is started? It was created by Chris Wanstrath, P. J. Hyett, Tom Preston-Werner, and Scott Chacon to help groups of developers work collaboratively on big software projects Pull Request It et you tell others about changes you've pushed to a branch in a repository on GitHub
What company owns it now? Microsoft Corp Workflows Is a configurable automated process that will run one or more jobs
