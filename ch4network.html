Networks & the Internet
NETWORK TERMS DEFINITION
  • Network Classifications
  1. LAN:
  2. MAN:
  3. WAN:
  • Is a collection of devices connected together in one physical location, such as a building, office, or home
  • Is a computer network that connects computers within a metropolitan area, which could be a single large city, multiple cities and towns, or any given large area with multiple buildings
  • Is the technology that connects your offices, data centers, cloud applications, and cloud storage together
  1. Network Models
  1. Bus network:
  2. Star network:
  • Is a local area network (LAN) topology in which each node a workstation or other device is connected to a main cable or link called a bus
  • Is a local area network (LAN) topology in which all nodes personal computers (PCs), workstations or other devices are directly connected to a common central computer that is often referred to as a hub
  1. Protocols
  1. CSMA/CD Ethernet:
  2. CSMA/CD wifi:
  • Is a medium access control (MAC) method used most notably in early Ethernet technology for local area networking
  • Is a network protocol for carrier transmission
  1. Combininb Networks
  1. Repeater:
  2. Bridge:
  3. Switch:
  4. Router:
  5. Getaway:
  • Is a type of network node that amplifies incoming signals and rebroadcasts them over a wider area or higher network layer than the original signal
  • Is a class of network device designed to connect networks at Open Systems Interconnection (OSI) Level 2, which is the data link layer of a local area network (LAN)
  • Is a device in a computer network that connects other devices together
  • Is a device that connects two or more packet-switched networks or subnetworks
  • Is a network node used in telecommunications that connects two networks with different transmission protocols together
  1. Client/Server Model
  1. p2p: peer-to-peer:
  2. Distributed system:
  3. Cloud computing:
  • Is created when two or more PCs are connected and share resources without going through a separate server computer
  • Is a system whose components are located on different networked computers, which communicate and coordinate their actions by passing messages to one another
  • Refers to the delivery of computing services over the internet, including storage, databases, software, and analytics
  1. Internet Adressing
  1. Tier 1:
  2. Tier 2:
  3. Tier 3:
  • Is a network that can reach every other network on the Internet without purchasing IP transit or paying for peering
  • Is an Internet service provider which engages in the practice of peering with other networks, but which also purchases IP transit to reach some portion of the Internet
  • Is by definition primarily engaged in delivering Internet access to end customers
  1. IP Adress
  1. ICANN:
  2. DNS:
  3. TLD: Top-level Domain
  4. Country-code TLDs:
  • Translates human readable domain names to machine readable IP addresses
  • Is everything that follows the final dot of a domain name
  1. Internet Protocols
  1. FTP:
  2. HTTP:
  3. SMTP:
  • Is a standard network protocol used for the transfer of files from one host to another over a TCP-based network, such as the Internet
  • Is an application protocol for distributed, collaborative, hypermedia information systems that allows users to communicate data on the World Wide Web
  • Is a TCP/IP protocol used in sending and receiving email
  1. World Wide Web
  1. URL:
  2. Client-side activities:
  3. Server-side activities:
  • Is nothing more than the address of a given unique resource on the Web
  • Refers to everything in a web application that is displayed or takes place on the client
  • Everything that happens on the server, instead of on the client
  1. Internet Four-layer Software
  1. Application layer:
  2. Transport layer:
  3. Network layer:
  4. Link layer:
  • Is actually an abstraction layer that specifies the shared protocols and interface methods used by hosts in a communication network
  • Is responsible for ensuring that the data packets arrive accurately and reliably between sender and receiver
  • Is the part of the Internet communications process where these connections occur, by sending packets of data back and forth between different networks
  • Is the lowest layer in the Internet protocol suite, the networking architecture of the Internet
  1. Internet Is
  1. Packet-switching network:
  2. Fault tolerant:
  3. Redundant:
  • Is the transfer of small pieces of data across various networks
  • Is a process that enables an operating system to respond to a failure in hardware or software
  • Is the process of providing multiple paths for traffic so that data can keep flowing even in the event of a failure
  1. Cyber Security
  1. Malware:
  2. Virus:
  3. Worm:
  4. Trojan horse:
  5. Spyware:
  6. Phishing:
  7. DDoS attack:
  • Is a file or code, typically delivered over a network, that infects, explores, steals or conducts virtually any behavior an attacker wants
  • Is a type of malicious software, or malware, that spreads between computers and causes damage to data and software
  • Is a type of malware whose primary function is to self-replicate and infect other computers while remaining active on infected systems
  • Is a type of malware that disguises itself as legitimate code or software
  • Is malicious software that enters a user's computer, gathers data from the device and user, and sends it to third parties without their consent
  • Is a technique for attempting to acquire sensitive data, such as bank account numbers, through a fraudulent solicitation in email or on a web site, in which the perpetrator masquerades as a legitimate business or reputable person
  • Is a cybercrime in which the attacker floods a server with internet traffic to prevent users from accessing connected online services and sites
